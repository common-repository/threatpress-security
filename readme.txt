== Description ==

ThreatPress is a free and simple, super-fast security and monitoring plugin that protects and scans your site for known malware, vulnerabilities, blacklisting, spam injections and more.

**Awesome Features**

* Checks your site's plugins, themes and WordPress core against our database of vulnerabilities [ThreatPress WordPress Vulnerabilities Database](https://db.threatpress.com)
* Scans your site for WordPress core changes, known malware, blacklisting status, phishing, injected spam from Google Safe Browsing servers, PhishTank, Malware domain list and Spamhaus ZEN. [ThreatPress SiteScan](https://sitescan.threatpress.com)
* Scheduled scans
* Email alerts
* Site diagnostic: Analyzes your site for weak security points
* Anti-Spam, 100% compatible with our [Google Invisible reCaptcha by ThreatPress](https://wordpress.org/plugins/invisible-recaptcha-by-threatpress)
* Login protection to lock out brute force hacks
* Password expiration
* Compatible with other security plugins
* Super lightweight, great performance

== Installation ==
1. Upload `threatpress` to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
